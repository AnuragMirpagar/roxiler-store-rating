# Store Rating Platform - Roxiler Coding Challenge 
 
## Tech Stack 
- Frontend: React.js 
- Backend: NestJS 
- Database: MySQL 
 
## Default Login Credentials 
Admin: admin@roxiler.com / Admin@123 
Normal User: user@test.com / User@1234 
Store Owner: owner@test.com / Owner@1234 
 
## How to Run 
1. Create MySQL database: roxiler_db 
2. cd backend, npm install, npm run start:dev 
3. npx ts-node src/seed.ts 
4. cd frontend, npm install, npm run dev
